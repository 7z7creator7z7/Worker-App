# Worker-App
📘 Worker Mini App — ish kunlaringizni belgilash, bonuslarni avtomatik hisoblash va xarajatlarni boshqarish uchun yaratilgan qulay ilova. Har 7 ta ish kuni uchun 100 000 so'm balansga qo'shiladi. Balansingiz orqali xarajatlarni qayd etishingiz, haftalar tarixini ko'rishingiz va barcha ma'lumotlarni bir joyda kuzatishingiz mumkin.
